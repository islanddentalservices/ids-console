Password Tips

• To edit website content you'll need your Github password.

• What is Github? It's the company that hosts the {% include config/target-website-name %} website.

• Please keep your Github password secret and safe - it could be used to vandalise or take over the {% include config/target-website-name %} website.

• You will also need passwords if you want to access social media or analytics accounts from here.

• If your web browser is set up to remember passwords, you may not need to enter them here.

Content Editor Tips

• It may take Github up to an hour to update the {% include config/target-website-name %} website to show the changes you made. A few minutes is more usual!

• You can be relaxed about the wait - if something is wrong and Github can't apply your changes, they'll email you immediately.
